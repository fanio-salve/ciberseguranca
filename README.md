# ciberseguranca
Análise de Atividades e Comportamentos de Ataques Cibernéticos

====================================================================================
Este projeto de Análise de Atividades e Comportamentos de Ataques Cibernéticos contempla:

Visão geral do projeto e objetivos analíticos;

Contextualização das tabelas e problemas de qualidade de dados;

Documentação detalhada de cada query, com propósito técnico e valor analítico;

Conexão clara entre as métricas extraídas e decisões em cibersegurança;

Linguagem adequada para auditoria técnica, governança de dados e apresentação a stakeholders.
